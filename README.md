# awsmec2
A start/stop/restart/snapshot CLI for Amazon EC2 instances

## Usage
```Usage: awsmec2.py [OPTIONS] COMMAND [ARGS]...

Options:
  -id, --instance-id TEXT
  -v, --verbose
  -d, --debug
  --help                   Show this message and exit.

Commands:
  backup
  reboot
  shutdown
  startup
  ```
